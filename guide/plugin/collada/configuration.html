# 配置文件

玩家模型配置文件暂时无法自动生成，您需要手动创建模型配置文件

Collada模型配置文件请放置于OrangeEngine/model/collada内

BlockBench模型配置文件请放置于OrangeEngine/model/blockbench内

示例文件

PlayerModel:
  # 模型名，请与 OrangeEngine/collada
  Name: "biped"
  # 盔甲模型
  ArmorModel:
    Head: "helmet_default"
    Chest: "chestplate_default"
    Legs: "leggings_default"
    Feet: "boots_default"
  # 骨骼绑定
  BindJoint:
    Head: "Head"
    MainHand: "Tool_R"
    OffHand: "Tool_L"
  Animation:
    Internal:
      Idle: "idle"
      IdleWithShift: "kneel"
      Death: "death"
      Walk: "walk"
      Jump:  "jump"
      Run: "run"
      Sneak: "sneak"
      Fly: "fly"
      Mount: "mount"
      Sleep: "sleep"
      Climb: "climb"
      Swim: "swim"
      Float: "float"
      Fall: "fall"
      Eat: ""
      Drink: ""
      Dig: "dig"
      BowAim: "bow_aim_mid"
      Block: "shield"
    # 移动动画列表（功能暂时移除）
    Custom:
      Movement: []

# 模型动画

# 第一人称

玩家模型的第一人称模型需要在第三人称模型名的基础上添加后缀_firstperson，例如biped与biped_firstperson。

# 镜像动画

在玩家在主副手使用物品时，您需要提供一个镜像动画文件，并以xxx.dae与xxx_mirror.dae命名，Mod将自动判断播放不同动画。

# 从模型软件导出Collada模型文件

# Blender

  1. 首先使用Blender软件打开blend模型文件

  1. 在文件 - 导出 - Collada(.dae)打开导出界面。

  1. 在右侧点击动画设置
  • 勾选包含动画选项
  • 帧类型选择采样
  • 采样率调大即可，OrangeEngine只需动画的关键帧即可
  • 勾选保持关键帧选项
  • 点击导出COLLADA按钮

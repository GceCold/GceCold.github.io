# 插件API

在群文件内你可以下载独立打包的插件API包 Javadocopen in new window

通过API你可以做到：

  • 绑定实体模型
  • 指定实体播放动作
  • 向不同玩家显示同一个实体不同模型
  • 向不同玩家播放不同的动画
  • 动态修改实体模型骨骼
    • 是否可视
    • 偏移
    • 旋转
    • 缩放
  • 监听客户端多种事件

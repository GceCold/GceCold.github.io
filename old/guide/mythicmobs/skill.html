# 技能机制[Mechanics]

本页作者: 小学生

OrangeEngine为MythicMobs提供了技能标签并且与ModelEngine高度匹配，兼容大部分来自ModelEngine的技能配置。

技能机制描述
Model为目标应用/移除指定模型
State为指定目标切换模型的状态[动画，模型]
Tint将模型的指定骨骼贴图进行着色
SwapModel更换实体模型
ChangePart更改模型的骨骼为设定值
SubModel向目标模型添加或删除不同模型的一部分
Petrify创建模型的雕像，此模型不能够更改或移动
DefaultState设置模型的默认状态，[就算不设置，依然存在默认动画]
SetModelTag设置模型名称标签上显示的名称
SetModelTagVisible设置模型名称标签的可见性
Disguise给目标玩家绑定模型
Undisguise移除目标玩家绑定的模型
LockHead锁定模型的俯仰和偏航旋转
PartVisibility设置模型一部分骨骼的可见性

# 技能机制

Model 为目标应用/移除指定模型

属性别名/简写说明默认
modelidm, mid, model应用指定的模型[名称]
hitboxh是否替换生物碰撞箱为配置文件中的设置]true
nametagn, name是否显示生物名字标签true
lockpitchlp, lpitch锁定头部俯仰角false
lockyawly, lyaw锁定头部偏航角false
remover移除模型false
invisiblei, invis设置实体是否不可见false
# MythicMobs实体生成后为自身应用名为"test"的模型, 同时启用名称标签"同步mm设置的名称"
- model{m=test;n=true} @self ~onSpawn

states切换后的状态[动画]
# 攻击时令自身所应用模型"test" 播放名为"attack"的动画

我是可爱的小绵羊:
  Type: HUSK
  Health: 20
  Damage: 0 #将生物的基础攻击设置为0，使用技能来替换普通攻击

Tint 将模型的指定骨骼组贴图进行着色

属性别名/简写说明默认
modelidmodel, mid, m所影响的模型名称
partidp, pid, part骨骼组名
colorc十六进制RGB/RGBA颜色，支持调整透明度 例如：FFFFFF/#FFFFFFFFFFFF
# 攻击后令自身应用模型"test"的名为"h_head"的骨骼组" 染为黑色
- tint{m=test;pid=h_head;c=000000} @self ~onAttack
1
2

SwapModel 更换实体模型

属性别名/简写说明默认
modelidmodel, mid, m所影响的模型名称
newmodelidp, pid, part新模型
# 受击且自身生命低于最大生命的一半将替换"test"模型为"test-1"模型
- swapmodel{m=test;n=test-1} 2self ~ondamaged =50%
1
2

ChangePart 技能目标所应用的指定模型的指定骨骼组替换为另一个指定模型的指定骨骼组

属性别名/简写说明默认
modelidmodel, mid, m所影响的模型名称
partidp, pid, part被替换的模型的骨骼组
newmodelidp, pid, part用于替换的骨骼组的模型名称
newpartidp, pid, part用于替换的骨骼组
# 生成后将自身所应用模型"testA"的名为"h_head"的骨骼组替换为"testB"的"h_head"
- changepart{m=testA;pid=h_head;nm=testB;npid=h_head} @self ~onSpawn
1
2

SubModel 向目标模型添加或删除不同模型的一部分

属性别名/简写说明默认
modelidmodel, mid, m所影响的模型名称
partidp, pid, part被影响的模型的骨骼组
submodeliddelid sm, smid, submodel要添加的骨骼组所位于的模型中的名称
subpartidsp, spid, subpart要添加的骨骼组名称
remover是否为删除子模型骨骼false
# 受伤后将自身所应用模型 "test"模型中的"h_head"骨骼组移除
- submodel{model=test;subpart=h_head;remove=true} @self ~onDamaged
# 攻击后为自身所应用模型:"test"模型中的"test_righthand"骨骼组添加模型"sword"中的组"diamond_sword"
- submodel{model=test;part=test_righthand;submodel=sword;subpart=diamond_sworld} @self ~onAttack
1
2
3
4

Petrify 创建模型的雕像，此模型将不再能够更改或移动

属性别名/简写说明默认
modelidmodel, mid, m所影响的模型名称
# 死亡后在自身位置创建一个无法交互、移除的模型"test"
- petrify{m=雕像} @self ~onDeath
1
2

DefaultState 将技能目标所应用的指定模型的指定默认状态替换为模型内的指定状态（包含动作）

属性别名/简写说明默认
modelidmodel, mid, m所影响的模型名称
typet动作类型。类型：IDLE, WALK, JUMP, DEATH
states切换后的状态[动画]
# 受击后将自身所应用模型"test"的状态"walk"更改为状态"run"
- defaultstate{m=test;t=walk;s=run} @self ~onDamaged
1
2

SetModelTag 设置模型的名称标签

属性别名/简写说明默认
tagt名称或占位符
visiblev名称标签的是否可见true
# 受击时显示自身所应用模型的血量，基于碰撞箱高度
- setmodeltagvisible{tag=<caster.hp>;v=true} @self ~onDamaged
1
2

SetModelTagVisible 设置模型名称标签的可见性

属性别名/简写说明默认
tagt名称或占位符
visiblev名称标签的是否可见true
# 受击且自身生命低于最大生命10%时隐藏自身所应用模型的标签，基于碰撞箱高度
- setmodeltagvisible{v=false} @self ~onDamaged =10%
1
2

Disguise 给目标玩家绑定模型

属性别名/简写说明默认
modelidm, mid, model应用指定的模型[名称]
# 右键后为右键者应用模型"test"
- disguise{m=test} @trigger ~onInteract
1
2

Undisguise 移除目标玩家绑定的模型

属性别名/简写说明默认
UndisguiseUndisguise移除模型
# 受击后移除攻击者的模型
- undisguise @trigger ~ondamaged
1
2

LockHead 移除目标玩家绑定的模型

属性别名/简写说明默认
lockpitchlp, lpitch锁定头部俯仰角false
lockyawly, lyaw锁定头部偏航角false
# 被右键后锁定自身所应用模型的俯仰视角度
- lockhead{lp=true} @self ~onInteract
1
2

PartVisibility 设置模型一部分的可见性。 这不会删除或添加任何部分

属性别名/简写说明默认
modelidm, mid, model所影响的模型名称
partidp, pid, part所影响模型的骨骼组名称
visibilityv, visible骨骼组是否可见true
# 受击且自身当前生命低于最大生命20%时隐藏所应用模型"test"的组"h_head"
- partvisibility{m=test;p=h_head;v=false} @self ~onDamaged =20%

目标选择器描述
ModelPart获取骨骼位置

# 目标选择器

ModelPart 获取骨骼位置

属性别名/简写说明默认
modelidm, mid, model所影响的模型名称
partidp, pid, part所影响模型的目标骨骼组
xx轴偏移0
yy轴偏移0
zz轴偏移0

TIP

大部分情况，Z 轴为前/后，X 轴为右/左，Y 轴为上/下
